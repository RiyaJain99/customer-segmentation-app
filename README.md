# 🛍️ AI Customer Segmentation Dashboard

A Machine Learning web application built using **Streamlit** and **K-Means Clustering** to segment mall customers based on their Annual Income and Spending Score.

## 🚀 Features

- Customer Segmentation using K-Means Clustering
- Interactive Streamlit Dashboard
- Beautiful Modern UI
- Customer Group Prediction
- Business Recommendations
- Download Customer Report

## 📊 Dataset

- Mall Customers Dataset
- 200 customer records
- Features:
  - Annual Income (k$)
  - Spending Score (1–100)

## 🧠 Machine Learning

- Algorithm: K-Means Clustering
- Number of Clusters: 5
- Data Standardization using StandardScaler

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
