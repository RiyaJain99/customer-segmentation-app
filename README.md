# 📖 About The Project

Customer churn is one of the biggest challenges faced by banks and subscription-based businesses.

This project leverages a **Random Forest Machine Learning model** to predict whether a customer is likely to leave the bank based on customer demographics and account information.

The model is deployed through an interactive **Streamlit Dashboard**, enabling real-time predictions for business users.

# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/RiyaJain99/customer-churn-prediction.git
```

Go into project

```bash
cd customer-churn-prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Streamlit

```bash
streamlit run app.py
```
